> 新笔记本电脑 Win11 跳过联网激活
>
> 参考链接：<https://www.bilibili.com/read/cv24265771/?from=articleDetail>

## 跳过激活

1. 新开机以后，一直到联网页面，注意 **先不要连接网络**

2. 按住键盘上的 `Shift+F10（或 Shift+Fn+F10）`，唤出命令行后输入 `OOBE\BYPASSNRO`，回车后自动重启

3. 重新回到联网界面，底下会出现【我没有 Internet 连接】，选择后进行下一步

4. 点击【继续执行受限设置】，点击后即可跳过联网激活！

## 再次激活

上述确认新笔电无误后，则需要重新联网激活，下述则通过系统自带工具实现再次激活，无需要还原出厂设置再次激活

1. `Win+r` 输入 cmd 打开命令符

2. 再输入 `cd %WINDIR%\system32\sysprep` 进入系统准备工具目录

3. 再次输入 sysprep

4. 系统跳出 **系统准备工具**

5. 选择 **进入系统全新体验**，关机选顶选择 **重新启动**，确认即可重启再次进入之前跳过的激活页登录微软账号完成激活
